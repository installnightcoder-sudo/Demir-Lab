# Contributing to Demir-Lab

Thank you for contributing to Demir-Lab!

## What Can Be Added?

* Guitar tones
* Bass tones
* Amp settings
* Pedal chains
* Music gear information

## Rules

* Be respectful.
* Use clear formatting.
* Include the gear used.
* Do not submit spam.
* Do not submit AI-generated nonsense.
* Make sure information is accurate.

## Pull Requests

Feel free to open a pull request with:

* New tones
* Improvements to existing tones
* Documentation improvements
* Formatting improvements
